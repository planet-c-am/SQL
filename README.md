# SQL
This repository contains multiple SQL modules designed for hands-on learning and practice. 

Each module focuses on a specific SQL topic, making it easy for beginners and intermediate learners 
to understand and practice queries step by step.

Purpose:

The goal of this repository is to provide structured SQL practice in a modular format. 
Each module is independent and can be run individually, allowing learners to gradually build 
their SQL skills while experimenting with realistic datasets and queries.

Audience:

- Beginners learning SQL fundamentals
- Students preparing for interviews or exams
- Anyone looking to practice writing SQL queries on sample datasets
